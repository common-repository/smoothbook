Smoothbook is an award winning appointments management system that embeds directly into your Wordpress website. 

Suitable for all types of business, Smoothbook works great for group and class bookings, has numerous features and is exceptionally easy to set up and use.

== Description ==
Smoothbook is a very popular, award winning online appointments system that manages thousands of appointments per day for businesses all over the world.

FEATURES

- Excellent for individual, group and class bookings
- Accept payments online: offer bulk discounts to your customers, offer discount and gift coupon codes
- Mailchimp integration: automatically add customers to Mailchimp newsletter lists
- Massively customisable: custom emails, registration / booking forms
- Looks fantastic: your booking calendar embeds seamlessly into your Wordpress page or post
- Very easy to set up and use
- London based limited company: your data is fully secure and protected under the UK data protection act.

== Installation ==
The most basic installation is a simple two step:

1. Upload the 'smoothbook' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the \'Plugins\' menu in WordPress

== Frequently Asked Questions ==
= Is Smoothbook really free? =
You can create a fully functional booking calendar and accept up to 5,000 appointments per month for free. Certain premium features require a subscription.

= Is Smoothbook suitable for groups and classes? =
We do a great deal of work organisations that offer group and class bookings

= Can I accept payments through Smoothbook =
Yes, the system allows you to offer as many payment options as you like: from single appointments to bulk discounts

= Is Smoothbook easy to use? =
Most of our users have their own professional booking calendar up and running within five minutes
